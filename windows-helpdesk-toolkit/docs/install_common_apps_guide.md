# Software Installation Guide with Chocolatey

## Purpose:
This script uses Chocolatey to install a set of common applications for new Windows setups or workstation onboarding.

## Requirements:
- Chocolatey must be installed. [Install Instructions](https://chocolatey.org/install)

## Applications Installed:
- Google Chrome
- 7-Zip
- Notepad++
- VLC Media Player
- Firefox

## How to Run:
1. Open PowerShell as Administrator.
2. Execute:
   ```powershell
   .\install_common_apps.ps1
